Instructions to compile and run the project for the judges
--------------------------------------------------------------------------------
Overview: This is a simple React App (next.js) with a node.js API running on the background.
Pre-reqs: You will need node.js and npm installed on your system

1. start the api (run the below commands from the root folder)

    cd api
    npm install
    npm run dev

2. On a different terminal, start the nextjs app (run the below commands from the root folder)

    cd my-app
    npm install
    npm run start:dev

3. Once both servers are running simultaneously on two different terminals,
   visiting localhost:3000 should load up the application
